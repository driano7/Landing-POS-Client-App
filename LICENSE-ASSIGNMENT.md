# Titularidad y cesión del código

Proyecto: `Landing-POS-Client-App`

Titular del proyecto: Donovan Riaño.

## 1. Alcance

Este documento describe la titularidad y el tratamiento de uso del código fuente, contenido, activos visuales e integraciones del sitio.

Todo derecho no expresamente cedido por escrito se entiende reservado.

## 2. Componentes de titularidad de Donovan Riaño

Se consideran de Donovan Riaño:

- contenido editorial y comercial
- branding, nombres comerciales y textos
- imágenes, íconos y activos visuales provistos para el sitio
- datos del negocio, precios, descripciones de servicios y casos
- rutas de aplicación, páginas y adaptadores específicos del sitio

## 3. Código reusable o de base técnica

Se consideran reutilizables, pero sujetos a cesión o licencia escrita a favor de Donovan Riaño:

- arquitectura de layout
- sistema SEO técnico
- builders de metadata y JSON-LD
- helpers de locale, navegación y composición visual
- componentes base del sistema visual
- animaciones distintivas y patrones de interacción

La reutilización externa de estos módulos requiere autorización escrita de Donovan Riaño.

## 4. Integraciones de terceros

Toda integración de terceros queda sujeta a la licencia, términos y condiciones del proveedor original, incluyendo:

- `next-themes`
- Radix UI
- Framer Motion
- Vercel Analytics
- WhatsApp links y assets de marca asociados

Si el proyecto incorpora un nuevo proveedor externo, su licencia debe revisarse de forma independiente.

## 5. Reglas de uso

- El contenido del cliente no puede reutilizarse sin autorización de Donovan Riaño.
- El código reusable no puede atribuirse a terceros sin respaldo documental.
- Cualquier cesión o licencia futura debe indicar alcance, vigencia, territorio y permisos de modificación, distribución y sublicencia.

## 6. Revisión legal

Cuando no exista contrato, cesión o acuerdo escrito verificable, la clasificación debe marcarse como:

- `requiere revisión legal/contractual`
